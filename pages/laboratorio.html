Cómo funciona el laboratorio

Cada experimento del Laboratorio Leonis toma un modelo de negocio concreto y lo enfrenta a dos perfiles humanos claramente distintos. Lo importante no es solo el modelo, sino qué ocurre cuando ese modelo entra en una vida real.

Paso 1

Se elige un modelo con una hipótesis interesante

Cada proyecto parte de una pregunta concreta: si este negocio puede sostenerse, qué fricciones genera y qué tipo de persona parece tener más encaje con él.

Paso 2

Se enfrentan dos perfiles con vidas distintas

No se comparan dos ideas. Se compara cómo la misma idea se comporta bajo dos contextos vitales, económicos y psicológicos diferentes.

Paso 3

Se documenta el proceso día a día

Cada perfil deja un diario narrativo donde aparecen cansancio, bloqueos, pequeñas decisiones, errores, avances y señales de encaje o fricción.

Qué hace distinto a este sistema

No simula solo negocios. Simula personas intentando construirlos.

El mismo modelo puede parecer una oportunidad razonable para un perfil y una mala decisión para otro. El laboratorio existe para observar precisamente esa diferencia: no solo si una idea funciona, sino para quién, bajo qué condiciones y a qué coste vital.

Tiempo ¿Cuánto margen real tiene esta persona para avanzar?
Capital ¿Puede asumir los costes sin poner en riesgo su estabilidad?
Energía ¿Tiene cabeza y cuerpo para sostener el esfuerzo?
Psicología ¿Cómo se relaciona con el riesgo, la paciencia y la incertidumbre?
Encaje ¿Este negocio mejora su vida o la tensiona todavía más?
